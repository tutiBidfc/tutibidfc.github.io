## 📖 How to Use

After installation, you can begin using the application. Follow these steps:

1. Open the application from your Start Menu or Applications folder.
2. Load an ndarray (N-dimensional array) that you want to reshape.
3. Specify the number of singleton dimensions you want to add to the front of your array.
4. Click on the "Execute" button. The application will display a read-only view of the reshaped array.

### Example

If you have an ndarray like this:

```
[[1, 2],
 [3, 4]]
```

And you specify to add two singleton dimensions, the output will look like this:

```
[[[[1, 2],
  [3, 4]]]]
```
